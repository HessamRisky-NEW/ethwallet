# ethwallet

회원가입 후 이더리움 계정을 발급받고 로그인하여 계정의 잔액, 이더리움 전송을 하는 웹 페이지

### 사용방법
1. 소스 내려받고 서버 실행하는 법
```
$ git clone https://github.com/Hyun-Geol/ethwallet.git
$ cd ethwallet
$ npm i
$ npm start
```
